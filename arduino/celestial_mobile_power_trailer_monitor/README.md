# Arduino Sketch

This folder contains the initial hardware test sketch for the Celestial Electric Mobile Power Trailer Monitor.

## Required libraries

Install through Arduino Library Manager:

- Adafruit BME280 Library
- Adafruit Unified Sensor

## Board

Recommended board:

- Arduino UNO R4 WiFi

## Serial Monitor

Set Serial Monitor to:

```text
115200 baud
```

## Cloud conversion

This sketch is a bench-test starting point. For Arduino Cloud:

1. Create a Thing in Arduino Cloud.
2. Add variables such as `temperatureF`, `humidityPercent`, `waterDetected`, `doorOpen`, and `systemAlert`.
3. Copy the sensor logic into the Cloud sketch generated by Arduino.
4. Test each sensor before field installation.
