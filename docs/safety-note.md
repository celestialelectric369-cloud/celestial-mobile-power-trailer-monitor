# Safety Note

This project is for low-voltage monitoring, training, and demonstration only.

Do not connect an Arduino directly to service equipment, PV strings, battery terminals, inverter terminals, generator controls, transfer switches, utility equipment, fire alarm circuits, or any line-voltage wiring.

Any real field installation must use proper listed equipment, isolation, overcurrent protection, enclosures, strain relief, wire separation, and code-compliant wiring methods. Dry-contact monitoring, relay interfaces, current sensors, and auxiliary signals must be installed according to the equipment manufacturer’s instructions and all applicable electrical codes.

This project is not a replacement for listed safety controls, manufacturer monitoring systems, utility-required protection equipment, fire alarm systems, generator/ATS controls, battery management systems, or code-required protective devices.

Celestial Electric uses this project as a low-voltage monitoring concept, training tool, and demonstration platform only.
